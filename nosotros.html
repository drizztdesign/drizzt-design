El estudio

Diseñamos webs como se diseñaba antes: con cuidado.

Manifiesto.

La web profesional ha perdido alma. Generadores con IA, plantillas de WordPress recicladas, agencias que entregan el mismo diseño cambiando solo el logo.

Drizzt Design existe para devolverle a las webs lo que tenían antes: identidad propia, código limpio, atención al detalle. Cada cliente recibe un sitio único, construido a mano, optimizado al milímetro.

No vendemos "soluciones". Vendemos tiempo y oficio.

Principios

Cómo trabajamos.

01

Sin frameworks innecesarios

HTML, CSS y JavaScript puro. La web debe poder leerse en 50 años con el mismo navegador. Si no necesitas React, no lo añadimos.

02

Mobile-first, sin excepciones

El móvil es el navegador real de tus visitantes. Diseñamos primero para móvil, luego escalamos. No al revés.

03

Velocidad como feature

PageSpeed mobile mínimo 90. Imágenes optimizadas, fuentes precargadas, scripts diferidos. La velocidad es parte del diseño.

04

Diseño editorial

Tipografía cuidada, espacios generosos, jerarquías claras. Las webs deben leerse como una buena revista, no como un panfleto.

05

Contenido real desde el día uno

Nada de Lorem ipsum. Si el cliente no tiene fotos o textos, paramos hasta tenerlos. Nunca diseñamos sobre ficción.

06

Auto-deploy continuo

Cada cambio se publica solo en Vercel vía GitHub. Sin FTPs, sin "sube esto al servidor". El cliente ve la web actualizada al instante.
