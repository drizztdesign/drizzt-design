Servicios

Lo que hacemos.

01 / Diseño web

Diseño de la web

Identidad visual, paleta, tipografías y maquetación a medida. Diseño editorial, no plantillas. Cada sección pensada para quien la va a leer.

02 / Desarrollo

Desarrollo frontend

HTML, CSS y JavaScript puros. Sin frameworks innecesarios. Animaciones y efectos cuidados. Mobile-first siempre.

03 / SEO

SEO técnico

Meta tags, Open Graph, Schema.org, sitemap y robots configurados. Listo para Google Search Console desde el día uno.

04 / Deploy

Deploy y mantenimiento

Auto-deploy GitHub → Vercel. Cada cambio en producción en 30 segundos. Mantenimiento por horas o paquete mensual.
